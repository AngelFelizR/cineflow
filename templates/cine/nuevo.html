Información Importante
  • Un cine puede tener múltiples salas de diferentes tipos.
  • No se permiten cines con el mismo nombre ni dirección.
  • El teléfono debe ser único para cada cine.
  • El iframe de Google Maps es esencial para que los clientes encuentren el cine.
  • Antes de eliminar un cine, asegúrate de que no tenga salas activas.
  • Se recomienda mantener todos los cines activos para ventas.
